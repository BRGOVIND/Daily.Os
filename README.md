## Why I built this

Every productivity app I tried eventually became overwhelming.

Some had too many features.
Some wanted subscriptions.
Some wanted me to create an account.
Most of them felt like project management software instead of something I'd actually enjoy opening every morning.

I wanted something different.

A planner where every day has its own workspace.

Open the app.
Click today's date.
Plan your day.
Get things done.
Close it.

No cloud.
No login.
No ads.
No AI chat popping up everywhere.

Just a clean place to organize your day.

---

## Features

- 📅 Calendar-first workflow
- ✅ Daily tasks
- 🎯 Today's Focus
- 🌸 Habit tracking
- 🔁 Recurring tasks
- 📝 Daily journal & notes
- 📊 Productivity insights
- 🌳 Progress tree
- 📦 Export / Import data
- 🌐 Offline-first
- 📱 Installable as a PWA

Everything is stored locally using IndexedDB.

Your data never leaves your computer.

## Running locally

```bash
git clone https://github.com/BRGOVIND/Daily.Os.git

cd Daily.Os

npm install

npm run dev
```

Then open

```
http://localhost:3000
```

## Philosophy

I don't want Daily OS to become another Notion clone.

The goal is simple.

Open it in the morning.

Plan your day.

Finish your work.

Come back tomorrow.

That's it.
